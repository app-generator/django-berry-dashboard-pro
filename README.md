## Manual Build 

> ðŸ‘‰ Download the code  

```bash
$ git clone https://github.com/app-generator/django-berry-dashboard-pro.git
$ cd django-berry-dashboard-pro
```

<br />

> ðŸ‘‰ Install modules via `VENV`  

```bash
$ virtualenv env
$ source env/bin/activate
$ pip install -r requirements.txt
```

<br />

> ðŸ‘‰ Edit the `.env` using the template `.env.sample`. 

```env

# True for development, False for production
DEBUG=True

```

<br />

> ðŸ‘‰ Set Up Database, you have to specify the database name `postgres_default` or `fallback_db`

```bash
$ python manage.py makemigrations --database=postgres_default
$ python manage.py migrate --database=postgres_default
```

<br />

> ðŸ‘‰ Create the Superuser, you have to specify the database name `postgres_default` or `fallback_db`

```bash
$ python manage.py createsuperuser --database=postgres_default
```

<br />

> ðŸ‘‰ Start the app

```bash
$ python manage.py runserver
```

At this point, the app runs at `http://127.0.0.1:8000/`.
